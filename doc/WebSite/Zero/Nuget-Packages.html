ASP.NET Boilerplate module-zero is distributed on nuget. Here, a list of all packages.

Abp.ZeroCore.IdentityServer4.EntityFrameworkCore

Entity Framework Core integration package for Abp.ZeroCore.IdentityServer4 package.

Abp.Zero

Main package of module zero.

Abp.Zero.Owin

Abp.Zero.AspNetCore
